# Structurizr GitHub Actions Setup

This bundle includes:
- Java exporter (`StructurizrWorkspace.java`) to generate `workspace.json`
- Maven `pom.xml` for dependencies and build
- GitHub Actions workflow (`pages.yml`) to export diagrams and publish to GitHub Pages

## Steps
1. Commit these files to your repo.
2. Push to `main` branch.
3. GitHub Actions will build the workspace, export diagrams (PNG/SVG), and deploy to Pages.
